## 主要发现

### 1. 登录模块
- 100% 通过。错误提示与实际场景匹配度高，锁定用户机制有效。
- SQL 注入和 XSS 测试均未成功绕过（系统有一定基础防护）。

### 2. 商品列表模块
- 核心功能（展示、添加、移除排序）均正常。
- **发现**：排序下拉框选中后未高亮选中项（BUG-001），影响用户体验。

### 3. 购物车模块
- 增删、导航功能正常。
- **发现**：登出后购物车被清空（BUG-002），对真实电商场景不友好。

### 4. 结算模块
- 表单字段非空校验正常工作。
- **发现**：Postal Code 不校验格式（接受字母和特殊字符），实际电商应加强校验（BUG-003）。

## 改进建议

1. 建议第三方登录持购物车数据（Cookie / LocalStorage）
2. Postal Code 增加格式校验（正则表达式匹配）
3. 排序下拉增加选中状态视觉反馈
